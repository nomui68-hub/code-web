# V13.1 - Tự luận đúng nghĩa

Bổ sung 3 kiểu tự luận:

```latex
\essayans{2|2,0}
```
Tự chấm theo kết quả cuối.

```latex
\essaymanual[2]
```
Học sinh nhập lời giải, giáo viên chấm tay trên dashboard.

```latex
\essayrubric{
  {0.5}{Lập đúng phương trình}
  {0.5}{Tính đúng kết quả}
}
```
Học sinh nhập lời giải, giáo viên chấm theo thang điểm từng ý.
